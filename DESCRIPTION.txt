Package: raTools
Type: Package
Title: A Toolbox for EI Research Associates
Version: 0.1-0
Author: Andrew Linville <andlinville@gmail.com>
Maintainer: Andrew Linville <andlinville@gmail.com>
Description: A set of tools to make the lives of EI Research Associates
	easier, and to make R a more approachable alternative to Stata.
	In particular, this package provides functions to create and update
	most of the standard outputs for a hospital case (patient origin tables,
	service area HHIs, distance matrices, overlap msdrg tables, etc.) as well
	as a set of other ad hoc tools to simplify standard tasks.
Depends: R (>= 3.0.0), plyr, reshape
